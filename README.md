SteamWolf 5.2.1 - Idler | Farmer | Claimer
Today I'm releasing SteamWolf — a modern Steam automation utility built with Next.js (frontend) and Tauri/Rust (backend), designed to automate and manage your Steam library efficiently while remaining lightweight and user-friendly.

🌟 Features 🌟
🎴 Card Farming
Automatically farm trading card drops from eligible games in your Steam library.

🏆 Achievement Manager
Unlock or relock achievements individually or in bulk with customizable delays and ordering options. Includes backup and restore functionality for achievement states.

⏱️ Playtime Booster
Simulate gameplay activity for up to 32 games simultaneously to increase recorded playtime hours.

📦 Inventory Manager
Manage your Steam inventory with bulk listing support, duplicate card detection, and automatic market price retrieval.

🎮 Automatic Free Game Claiming
Automatically detect and claim free Steam games and limited-time giveaways as they become available.

🎨 Modern Interface
Clean, ad-free interface featuring the custom SteamWolf purple-blue theme.

🚀 Startup & Tray Support
Launch automatically with Windows and run quietly from the system tray.

⚙️ Technology
SteamWolf uses a Next.js frontend combined with a Tauri/Rust backend for performance and low resource usage. Certain Steam-related background operations are handled through bundled native utilities, allowing the application to remain lightweight and responsive.

All operations are performed locally on the user's machine. No Steam account credentials are stored on external servers.

🔒 Security
SteamWolf does not modify game files, inject into running games, or alter Steam client files. The application operates externally and focuses exclusively on Steam library management, inventory automation, achievement management, and playtime simulation.
